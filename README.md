# Fetch and download The Official UK top 40 single charts

## Usage

Just run this command


```bash
$ python musicbot.py
```


## Requirements
1. BeautifulSoup
2. Youtube-dl

Inspired from [instantmusic](https://github.com/yask123/Instant-Music-Downloader)
